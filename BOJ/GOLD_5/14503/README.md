# 로봇 청소기

[백준 14503번: 로봇 청소기](https://www.acmicpc.net/problem/14503)

## 문제 파악하기

로봇 청소기가 주어진 규칙에 따라 이동하며 청소하는 칸의 개수를 구하는 **시뮬레이션** 문제입니다.

### 로봇 청소기 동작 규칙

1. 현재 칸이 청소되지 않았으면 청소
2. 왼쪽 방향으로 회전하면서 4방향 확인
   - 청소되지 않은 빈 칸이 있으면 전진
3. 4방향 모두 청소되었거나 벽이면 후진 (방향 유지)
4. 후진할 곳도 벽이면 종료

> [!NOTE]
> 방향 표현: 북(0), 동(1), 남(2), 서(3)
>
> - **왼쪽 회전**: `(d + 3) % 4` (북→서→남→동→북)
> - **후진**: `(d + 2) % 4` 방향으로 이동하되 **방향은 유지**

## 풀이

### 1. 방향 배열 정의

```javascript
const dx = [-1, 0, 1, 0]; // 북 동 남 서
const dy = [0, 1, 0, -1];
```

### 2. 현재 칸 청소

```javascript
if (board[x][y] === 0) {
  board[x][y] = 2; // 청소 완료 표시
  answer++;
}
```

> [!TIP]
> 청소한 칸을 **2로 표시**하는 것이 핵심!
>
> - `0`: 청소되지 않은 빈 칸
> - `1`: 벽
> - `2`: 청소 완료
>
> 이렇게 하면 "청소 안 된 빈 칸"과 "청소된 칸"을 구분할 수 있습니다.

### 3. 왼쪽 회전하며 탐색

```javascript
for (let i = 0; i < 4; i++) {
  d = (d + 3) % 4; // 왼쪽 회전
  const nx = x + dx[d];
  const ny = y + dy[d];

  if (board[nx][ny] === 0) {
    return getResult(nx, ny, d); // 청소 안 된 칸으로 전진
  }
}
```

4방향을 왼쪽으로 회전하면서 확인하고, 청소되지 않은 빈 칸이 있으면 그 방향으로 전진합니다.

### 4. 후진 처리

```javascript
const backDir = (d + 2) % 4;
const bx = x + dx[backDir];
const by = y + dy[backDir];

if (board[bx][by] === 1) return; // 뒤가 벽이면 종료
return getResult(bx, by, d); // 방향은 유지하고 후진
```

- 4방향 모두 이동 불가하면 후진
- 후진 방향은 `(d + 2) % 4`이지만, **로봇의 방향 d는 유지**
- 뒤가 벽이면 작동 종료

### 5. 결과 출력

```javascript
getResult(sx, sy, d);
console.log(answer);
```

시작 위치와 방향으로 시뮬레이션을 실행하고 청소한 칸의 수를 출력합니다.
