# 타일 채우기

[백준 2133번: 타일 채우기](https://www.acmicpc.net/problem/2133)

## 문제 파악하기

3×N 크기의 벽을 **2×1, 1×2 타일**로 채우는 경우의 수를 구하는 문제입니다.

### 핵심 관찰

- N이 **홀수**면 경우의 수는 **0** (면적이 홀수라 2칸 타일로 채울 수 없음)
- N이 짝수일 때만 경우의 수 존재

> [!NOTE] > **DP** 문제입니다. 점화식을 유도하는 것이 핵심!

## 풀이

### 기본 케이스

```javascript
const dp = Array(N + 1).fill(0);
dp[2] = 3;
dp[4] = 11;
```

```
dp[2] = 3가지:
┌──┬──┐  ┌──────┐  ┌──────┐
│  │  │  ├──────┤  ├──────┤
│  │  │  ├──────┤  ├──────┤
└──┴──┘  └──────┘  └──────┘
```

### 점화식

```javascript
for (let i = 6; i <= N; i++) {
  if (i % 2 === 1) continue;
  dp[i] = 3 * dp[i - 2] + (dp[i - 2] - dp[i - 4]);
}
```

이 점화식은 최적화된 형태입니다. 전개하면:

```
dp[i] = 4 * dp[i-2] - dp[i-4]
```

> [!TIP] > **점화식 유도**
>
> 원래 점화식: `dp[n] = 3*dp[n-2] + 2*(dp[n-4] + dp[n-6] + ... + dp[0])`
>
> - 3가지 기본 패턴으로 2칸을 채우고 나머지 dp[n-2]
> - 각 짝수 길이마다 2가지 특수 패턴 존재
>
> 이를 **인접 항의 차이**로 정리하면: `dp[n] = 4*dp[n-2] - dp[n-4]`

## 특수 패턴 (2개씩)

```
4칸 특수:        6칸 특수:        8칸 특수: ...
┌──────────┐    ┌───────────────┐
├────┬─────┤    ├────┬────┬─────┤
├────┴─────┤    ├────┴────┴─────┤
└──────────┘    └───────────────┘
(+ 상하 대칭 1개씩)
```

## 동작 예시

```
dp[2] = 3
dp[4] = 11
dp[6] = 4*11 - 3 = 41
dp[8] = 4*41 - 11 = 153
```

## 핵심 정리

| N   | dp[N] | 계산      |
| --- | ----- | --------- |
| 1   | 0     | 홀수      |
| 2   | 3     | 기본      |
| 4   | 11    | 기본      |
| 6   | 41    | 4×11 - 3  |
| 8   | 153   | 4×41 - 11 |

### 왜 홀수는 0인가?

- 3×N에서 N이 홀수면 총 칸 수가 3N (홀수)
- 2×1 타일은 2칸을 차지 → 짝수 칸만 채울 수 있음

### 시간 복잡도

- **O(N)**
