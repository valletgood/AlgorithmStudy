## 문제

https://school.programmers.co.kr/learn/courses/30/lessons/12938

### 문제 파악하기

위 문제는 프로그래머스 최고의 집합 문제이다.

n개의 자연수로 이루어진 집합 중에서 합이 s이면서 각 원소의 곱이 최대가 되는 집합을 찾는 문제다.

> [!NOTE] > **핵심 수학적 원리**: 합이 일정할 때, 곱을 최대로 만들려면 숫자들이 최대한 균등해야 한다.
>
> 예를 들어, 합이 10이고 3개의 숫자가 필요하다면:
>
> - [3, 3, 4] → 곱: 36 ✅
> - [2, 4, 4] → 곱: 32
> - [1, 5, 4] → 곱: 20
>
> 가장 균등하게 분배한 [3, 3, 4]가 곱이 최대!

## 풀이

### 1. 불가능한 경우 처리

```javascript
if (s < n) return [-1];
```

- s가 n보다 작으면 n개의 자연수(최소 1)로 합을 만들 수 없으므로 [-1] 반환

### 2. 균등 분배

```javascript
const d = Math.floor(s / n); // 몫
let r = s % n; // 나머지

let dArr = Array.from({ length: n }).fill(d);
```

- `s / n`의 몫으로 배열을 모두 채움
- 이렇게 하면 가장 균등한 분배의 기본값이 됨

### 3. 나머지 분배

```javascript
if (r === 0) {
  return dArr;
}

let index = n - 1;
while (r > 0) {
  dArr[index] = dArr[index] + 1;
  r--;
  index--;
}
```

> [!TIP] > **나머지는 뒤에서부터 1씩 분배**
>
> - 나머지가 0이면 이미 균등 분배 완료
> - 나머지가 있으면 배열 뒤쪽부터 1씩 더해줌
> - 이렇게 하면 오름차순 정렬이 자연스럽게 유지됨
>
> 예: n=3, s=10 → 몫 3, 나머지 1
>
> - 초기: [3, 3, 3]
> - 나머지 분배: [3, 3, 4] ✅
